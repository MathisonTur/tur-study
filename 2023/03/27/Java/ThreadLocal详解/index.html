ThreadLocal详解

一、ThreadLocal 简介

ThreadLocal 叫做 线程变量，指 ThreadLocal 中填充的变量属于当前线程，该变量对于其他线程而言是隔离的，即该变量是当前线程独有的变量。ThreadLocal 为变量在每个线程中都创建了副本，那么每个线程都可以访问自己内部的副本变量
