IDEA中创建文件模板

IDEA

更新日期:   2023-04-10

阅读时长:   1 分

在 IDEA 中创建适合自己项目的文件模板，可以在新建文件时，免去一些常规重复的操作，如一些类中常添加的注解如@Data、@Builder等等，创建文件时，自动导入这些可以提高些许效率。

so， 开始吧。

下面以创建 pojo 模型类为例：

创建 pojo 模型类时，一般需要添加注解：

  • @Data
  • @Builder
  • @NoArgsConstructor
  • @AllArgsConstrustor

至于这些注解的用处，想必都知道

所以，目标便是将添加的注解放入到模板

步骤如下：

1. 打开：设置 -> 编辑器 -> 文件和代码模板

2. 在文件栏里新建文件

  • 名称的命名随意，建议文件名容易知道它的用处。

  • 文件扩展名默认 java。如果需要创建其他类型的需要改动

  • 文件名(F)：即新建的类文件的文件名。书写格式如下：$ClassName

    注：ClassName可以自定义，在新建文件时会出现的新建窗口，填写的值即文件名

3. 在编辑框里编写模板
  • 添加类的javadoc注释
  • 添加注解
  • 添加类的定义

内容如下：

import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;
import lombok.Builder;

/**
* @author ${USER}
* @date ${DATE} ${TIME}
* @description TODO
* @version 1.0
*/
@Data
@Builder
@AllArgsConstructor
@NoArgsConstructor
public class $ClassName  {

}

其中 $ClassName 需要与上方填写的文件名一致，保证文件名和类名一致

4.创建好的模板如下所示：

5. 模板的使用：

创建好的模板记得点应用

在新建选项可以看到新建的模板出现了：

点击新建的模板，弹出填写 ClassName（文件名）的窗口：

填写你需要的文件名，即可创建成功文件。
